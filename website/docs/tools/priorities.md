# Priorities

Priorities represent severity levels that can be assigned to incidents in PagerDuty.

## Tools

### `list_priorities`

List all priorities configured in the account.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| (none) | | | This tool takes no parameters |

**Example prompt:**

> "What priority levels are configured in our PagerDuty account?"

---

> "Show me all available incident priorities so I can set the right one"
